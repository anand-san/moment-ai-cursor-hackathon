# Mobile Native Experience

MomentAI is built for mobile-first usage, with Android support via Capacitor.

## Native Behaviors

- Device speech recognition for voice brain dumps
- Haptic feedback on key interactions (like swipes)
- Local notifications for timer/reminder tip actions
- Native SMS handoff for message-style tips
- Deep-link based auth flow support

## Tip Quick Actions

Depending on action type, a tip can:

- Start a timer
- Schedule a reminder
- Open messages with prefilled text
- Save itself locally for later

## Mobile UX Details

- Native back button behavior is handled explicitly
- Safe-area spacing is used for modern phone layouts
- App links are configured for `momentai.sandilya.dev`
