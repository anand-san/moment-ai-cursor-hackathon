See @README.md for project overview and @package.json for available Bun commands for this project.

The app uses Bun for package management and running scripts.

## Coding Workflow

1. Before starting work, pull latest changes
2. Run `bun install` if dependencies changed
3. Start dev servers with `bun run dev`
4. Before committing, run `bun run test`, `bun run lint`, and `bun run build`
5. For Android changes, validate with `bun run android:build`
